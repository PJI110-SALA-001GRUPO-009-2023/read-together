# Read Together Server

Servidor da nossa aplicação. Tecnologias usadas:

- Node (18.5.0 - LTS)
- NPM (9.5.0)
- ExpressJS
- TypeScript
- Prisma

## Instalação

Primeiro realize o `git clone` :

```sh
git@github.com:PJI110-SALA-001GRUPO-009-2023/read-together-server.git
```

Em seguida:

```sh
npm install
```
